# Ejecutar en Desarrollo

1. Clonar el repositorio
2. Ejecutar:
```
pnpm install
```

3. Tener Nest CLI instalado
```
pnpm install -g @nestjs/cli
```

4. Levantar la DB mediante Docker:
```
docker-compose up -d
```

5. Ejecutar la aplicaci√≥n:
```
pnpm run start:dev
```
